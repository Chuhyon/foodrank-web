---

## 1. 수익화 전제 조건

수익화를 시작하기 전 아래 두 지표를 동시에 충족해야 함.
하나라도 미달 시 수익화 착수를 연기하고 콘텐츠·UX 개선에 집중.

| 지표 | 기준값 | 근거 |
|------|--------|------|
| MAU | 5,000명 이상 | 광고주에게 제안 가능한 최소 트래픽 |
| D30 리텐션 | 15% 이상 | 사용자가 서비스 가치를 인정했다는 최소 기준 |
| 키워드 CTR | 15% 이상 | "클릭할 만큼 유용하다"는 사용자 행동 증거 |

---

## 2. 2단계 수익 모델 상세

### 모델 A — 브랜드 스폰서 노출

급상승 브랜드 탭에서 상위 노출 슬롯을 브랜드사에 유료 제공.

```
UI 구조:
  1  ↑↑  스폰서브랜드   [스폰서]   ← 월 단위 계약
  2  ↑↑  마라탕 브랜드
  3  →   자연 순위 브랜드
  ...

계약 조건:
  - 계약 기간: 월 단위 (최소 1개월)
  - 노출 위치: 급상승 브랜드 탭 고정 상단 1~2슬롯
  - [스폰서] 표기 필수 (광고 식별 의무)
  - 클릭 시 검색 연결은 동일하게 적용 (F2 방식)
```

**가격 책정 기준:**

| MAU | 슬롯당 월 단가 | 근거 |
|-----|--------------|------|
| 5,000 | ₩30~50만원 | 외식업 타겟 소규모 지면 기준 |
| 10,000 | ₩80~150만원 | 트래픽 2배 + 타겟 정교화 가치 반영 |
| 30,000 | ₩200~400만원 | 업계 주요 매체 수준 |

---

### 모델 B — 창업 광고 DB 리드

급상승 브랜드 TOP10에 진입한 브랜드 DB를 활용해
창업 희망자에게 "창업 정보 받기" CTA를 노출, 리드를 브랜드사에 연결.

```
흐름:
  사용자: 브랜드 클릭 → 구글 검색 후 복귀
      ↓
  2단계 대시보드 하단 CTA 노출:
  "📌 [브랜드명] 창업 정보가 궁금하다면? → 무료 상담 신청"
      ↓
  리드 수집 (이름, 연락처, 관심 지역)
      ↓
  브랜드사 전달 (리드 단가 계약)
```

**리드 단가 근거:**

| 항목 | 값 | 출처 근거 |
|------|-----|-----------|
| 외식 창업 리드 단가 | ₩10,000~50,000/건 | 프랜차이즈 창업 플랫폼 일반 시장 기준 |
| 전환율 가정 | 브랜드 클릭 사용자의 2% | 보수적 추정 (일반 창업 정보 사이트 1~3%) |
| MAU 5,000 기준 월 리드 수 | 약 20~40건 | 일 방문 500명 × 브랜드 탭 40% × 2% × 30일 |

---

### 모델 C — 디스플레이 배너 광고

외식업 관련 광고주(식자재 공급업체, 포스 시스템, 창업 플랫폼 등)의
배너 광고를 대시보드 하단 또는 탭 콘텐츠 하단에 노출.

```
형태:
  - 정적 이미지 배너 (728×90 리더보드 또는 320×100 모바일 배너)
  - 위치: 리스트 하단 고정 또는 탭 전환 시 1회 노출
  - 직접 광고 계약 방식 (Google AdSense는 외식업 타겟 최적화 한계로 제외)

수익 방식:
  - CPM (노출 1,000회당 단가): ₩3,000~3,500 (외식업 타겟 버티컬 미디어 기준)
  - 월 정액 계약 가능 (광고주가 노출 예측을 선호하는 경우)
```

> 📌 배너 광고는 트래픽이 충분히 축적된 2단계 이후에 도입.
> MVP 단계에서는 광고 UI 요소 없음 — 온보딩·UX 완성도 우선.

## 3. 수익 시뮬레이션 계산 상세

### 공통 가정값

| 항목 | 값 | 근거 |
|------|-----|------|
| 외식업 타겟 배너 CPM | ₩3,000~3,500 | 국내 버티컬 미디어 타겟 CPM 평균 |
| 페이지뷰 / MAU | 4~6회 | 대시보드 특성상 반복 방문, 탭 전환 포함 |
| 리드 단가 | ₩15,000 (보수적 중간값) | 외식 창업 플랫폼 시장 기준 |
| 브랜드 클릭 → 리드 전환율 | 2% | 보수적 추정 |
| 스폰서 슬롯 단가 | MAU 기반 협상 | 모델 A 기준 |

### MAU 5,000 시뮬레이션

```
[디스플레이 배너 광고 — 모델 C]
  월 PV = 5,000명 × 4회 = 20,000 PV
  CPM ₩3,000 기준: 20,000 / 1,000 × 3,000 = ₩60,000

[스폰서 슬롯 1개 — 모델 A]
  단가: ₩300,000 (MAU 5,000 기준)

[창업 리드 — 모델 B]
  일 방문 167명 (= 5,000 / 30) × 브랜드 탭 40% = 67명
  × CTR 25% = 17명 클릭
  × 리드 전환 2% = 0.34명/일 → 월 10건
  × 리드 단가 ₩15,000 = ₩150,000

합계: 60,000 + 300,000 + 150,000 = ₩510,000 ≈ 약 50만원/월
```

### MAU 10,000 시뮬레이션

```
[디스플레이 배너 광고 — 모델 C]
  월 PV = 10,000 × 5회 = 50,000 PV
  CPM ₩3,000: 50,000 / 1,000 × 3,000 = ₩150,000

[스폰서 슬롯 2개 — 모델 A]
  단가: ₩800,000 (MAU 10,000 기준)

[창업 리드 — 모델 B]
  일 방문 333명 (= 10,000 / 30) × 40% × 25% × 2% = 0.67명/일 → 월 20건
  × ₩15,000 = ₩300,000

합계: 150,000 + 800,000 + 300,000 = ₩1,250,000 ≈ 약 150만원/월
```

### MAU 30,000 시뮬레이션

```
[디스플레이 배너 광고 — 모델 C]
  월 PV = 30,000 × 6회 = 180,000 PV
  CPM ₩3,500: 180,000 / 1,000 × 3,500 = ₩630,000

[스폰서 슬롯 3개 — 모델 A]
  단가: ₩2,400,000 (MAU 30,000 기준, 3슬롯)

[창업 리드 — 모델 B]
  일 방문 1,000명 (= 30,000 / 30) × 40% × 25% × 2% = 2명/일 → 월 60건
  × ₩20,000 (단가 상향) = ₩1,200,000

[창업 컨설팅 (3단계 시작)]
  월 3건 × ₩500,000 = ₩1,500,000

합계: 630,000 + 2,400,000 + 1,200,000 + 1,500,000 = ₩5,730,000 ≈ 약 500만원+/월
```

## 4. 3단계 수익 모델 상세

### 자사 창업 컨설팅 서비스

```
대상: 예비 창업자 (푸드랭크 사용자 중 창업 관심 CTA 클릭)
상품: 푸드랭크 트렌드 데이터 + 브랜드 DB 기반 1:1 창업 컨설팅

상품 구성:
  기본: ₩300,000 — 트렌드 데이터 리포트 + 1회 화상 상담 (1시간)
  표준: ₩700,000 — 맞춤 브랜드 비교 분석 + 2회 상담
  프리미엄: ₩1,500,000 — 창업 로드맵 + 3회 상담 + 사후 모니터링

운영 방식 (초기):
  외부 프랜차이즈 컨설턴트 협력사 모델 (수익 분배)
  자사 컨설팅 팀 구성은 MAU 50,000 이상 시 검토
```

### 식자재 발주 연결

```
대상: 현직 사장님 (물가 탭 사용자)
방식: 물가 주의보 탭 → 식자재 공급업체 연결 링크
수익: 공급업체로부터 CPC 또는 거래 수수료

예시 흐름:
  물가 탭 "대파 +35%" 클릭
    → 구글 검색 (기존 F2)
    → 대시보드 하단: "대파 저렴하게 구매하기" CTA
    → 제휴 식자재 공급업체 연결
    → 거래 성사 시 수수료 수취
```

## 5. 브랜드 DB 수익화 자산화 전략

```
1단계 (지금부터):
  급상승 브랜드 TOP10 진입 시 brand_db 자동 저장
  항목: 브랜드명, 카테고리, 최고순위, 최초등장일, 공개 연락처

2단계 (MAU 5,000 이후):
  브랜드 DB 활용 영업:
    "귀사 브랜드가 지난 주 급상승 1위를 기록했습니다"
    → 스폰서 노출 or 창업 리드 계약 제안
  GA4 타겟팅:
    keyword_click(브랜드 탭) 사용자 → 창업 광고 리타겟팅

3단계 (MAU 30,000 이후):
  브랜드 DB × 트렌드 데이터 → 컨설팅 상품 핵심 자료
  식자재 물가 DB × 사장님 세그먼트 → 발주 연결 상품
```
